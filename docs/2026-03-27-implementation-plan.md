# 个人知识库 — 实施计划

> 日期：2026-03-27
> 基于：`2026-03-27-personal-knowledge-base-design.md`

---

## Phase 1：脚手架 + 知识输入 + LLM 处理 + 存储

### 1.1 项目初始化
- [ ] pnpm monorepo 初始化（根 `package.json` + `pnpm-workspace.yaml`）
- [ ] `client/` — Vite + React + TypeScript 脚手架
- [ ] `server/` — Fastify + TypeScript 脚手架
- [ ] `shared/` — 共享类型定义目录
- [ ] 统一 tsconfig，路径别名配置
- [ ] 开发环境：Vite proxy → Fastify，热重载打通

### 1.2 数据库层
- [ ] better-sqlite3 集成，数据库初始化脚本
- [ ] 建表：`knowledge_items`、`knowledge_links`、`prompt_templates`、`review_records`、`categories`
- [ ] 封装基础 CRUD 操作（Repository 模式）
- [ ] 数据库 migration 机制（简单版本号即可）

### 1.3 LLM 适配层
- [ ] 统一 LLM 接口定义（`LLMProvider` 抽象）
- [ ] OpenAI 适配器实现
- [ ] 支持配置切换（API Key、模型名、base URL）
- [ ] 流式输出支持（SSE）
- [ ] 错误重试 + 速率限制处理

### 1.4 Prompt 模板引擎
- [ ] 内置 4 种模板：文言文、成语、诗词、通用
- [ ] 模板渲染引擎（变量替换 + JSON 输出解析）
- [ ] 模板 CRUD API
- [ ] 前端模板管理页面（简单列表 + 编辑）

### 1.5 知识输入管道
- [ ] 文件上传 API（支持 PDF、Markdown、图片）
- [ ] PDF 文本提取（pdf-parse）
- [ ] 图片 OCR（先走 LLM 视觉能力）
- [ ] 知识处理管线：输入 → 解析 → LLM 处理 → 存储
- [ ] 自动分类：LLM 识别类型 + 分类，支持静默/确认两种模式
- [ ] 关键词提取 + 标签生成

### 1.6 前端 — 知识输入
- [ ] 文本输入编辑器（Markdown 预览）
- [ ] 文件上传组件（拖拽 + 点击）
- [ ] 知识类型选择器（手动选 / 自动识别）
- [ ] 处理结果预览 + 确认/编辑
- [ ] 知识列表页（分页、搜索、筛选）

## Phase 2：知识图谱可视化

### 2.1 关联识别
- [ ] LLM 在处理新知识时，传入已有知识标题列表，识别关联
- [ ] 关联存储到 `knowledge_links` 表
- [ ] 关联类型和强度评估

### 2.2 图谱 API
- [ ] `GET /api/graph` — 返回所有节点和边
- [ ] `GET /api/graph/cluster` — 聚类结果
- [ ] 支持按分类/标签过滤图谱

### 2.3 聚类算法
- [ ] 基于标签/分类的简单聚类（先实现）
- [ ] Louvain 社区检测算法（可选优化）

### 2.4 前端 — 图谱可视化
- [ ] Cytoscape.js 集成，力导向图布局
- [ ] 节点样式：颜色按类型、大小按关联数
- [ ] 交互：缩放、拖拽、点击节点查看详情侧边栏
- [ ] 搜索定位节点
- [ ] 聚类着色切换

## Phase 3：Agent 问答

### 3.1 RAG 引擎
- [ ] 知识条目向量化（调用 OpenAI Embedding API）
- [ ] 向量存储（SQLite + sqlite-vss 或简单余弦相似度）
- [ ] 检索：查询 → embedding → 相似度匹配 → 返回 top-k
- [ ] 上下文组装：检索结果 + 用户问题 → LLM 生成回答

### 3.2 对话 API
- [ ] `POST /api/agent/chat` — SSE 流式输出
- [ ] 对话历史存储和加载
- [ ] 引用来源标注

### 3.3 前端 — Agent 对话
- [ ] 聊天 UI 组件（消息列表 + 输入框）
- [ ] 流式渲染（逐字显示）
- [ ] 引用来源可点击跳转到对应知识条目
- [ ] 对话历史切换

---

## Phase 4：复习系统

### 4.1 题目生成
- [ ] 根据知识类型生成不同题型
  - 文言文：翻译题、字词解释
  - 成语：填空、选择释义
  - 诗词：默写、赏析问答
  - 通用：关键词匹配、简答
- [ ] 题目存储到 `review_records`

### 4.2 复习调度
- [ ] 间隔重复算法（SM-2 简化版）
- [ ] `GET /api/review/today` — 今日待复习列表
- [ ] `POST /api/review/submit` — 提交答案 + 更新间隔

### 4.3 前端 — 复习
- [ ] 复习页面：题目展示 + 答题 + 即时反馈
- [ ] 复习进度统计（今日完成/总数、正确率）
- [ ] 复习日历（可选）
